# Linux and Open Source Concepts

Linux is a powerful platform that can be used in a wide range of devices, from smartphones to servers and supercomputers.

Linux is actually **a kernel**, not a complete operating system by itself. The kernel manages hardware resources and system operations.

Various tools and software are built on top of the Linux kernel, **many of which are provided by the GNU Project**, forming a complete operating system.

Linux is a **Unix-like system** that follows many Unix concepts and standards. As a result, many Unix commands and applications can run on Linux.

macOS is a Unix-based operating system that uses the **XNU kernel and Apple's proprietary graphical interface**.

Windows provides similar capabilities such as file, user, and process management, but it has a **different kernel, architecture, and vendor**.

Linux is available in many distributions, allowing users to choose the most suitable distribution for their needs.


# Software Licenses

A software license defines the rules for using, modifying, distributing, and redistributing software.

Open-source licenses also allow users to **view, modify, and redistribute the source code**.

Major organizations in this field:

1. Free Software Foundation (FSF)
2. Open Source Initiative (OSI)
3. Creative Commons (CC)

Each organization defines its own licensing principles and standards.



# Copyright

Copyright is the legal right to create, use, and distribute copies of a work.

Most copyright laws were originally created for books, music, films, and artistic works, and were later applied to software.


# Copying Software

In computing, many operations involve copying data:

* Copying files between storage devices
* Moving data from SSD to RAM
* Moving data from RAM to CPU cache
* Creating backups
* Transferring data between storage systems

**Many of these copies are necessary for normal software execution and system operation.**


# Open Source

Open source allows users to:

* View software
* Copy software
* Modify software
* Develop new versions
* Redistribute software


# FSF (Free Software Foundation)

The Free Software Foundation was founded in 1985 by **Richard Stallman**.

It created the GNU Project and promotes software freedom.

The FSF defines four essential freedoms:

1. Freedom to run the software for any purpose
2. Freedom to study the source code
3. Freedom to modify the software
4. Freedom to redistribute original or modified versions

**Free software does not necessarily mean free of charge.**

# OSI (Open Source Initiative)

OSI focuses on the practical and commercial aspects of open-source software.

Its goal is to make open-source software suitable for business and commercial use.

Key OSI principles include:

* Free redistribution
* Source code availability
* No discrimination against persons or groups
* No restrictions on fields of use
* Integrity of source code
* Automatic distribution of the license
